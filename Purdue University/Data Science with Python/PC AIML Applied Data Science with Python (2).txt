3) Core content reviewed and taught
- Creating a DataFrame from a Python dictionary:
  - Recognized a dictionary by curly braces and key:value pairs.
  - Values for keys were lists; keys become column headers when converted to a DataFrame using pandas.DataFrame().
  - DataFrame displays as a table with column headers and associated column data.

- Index-based access with DataFrame.iat:
  - iat(row_index, column_index) accesses a single cell using integer indexes for row and column.
  - Examples explained and demonstrated:
    - Access first row, second column (row index 0, column index 1) → expected value 10.
    - Access second row, second column (row index 1, column index 1) → expected value 20.
    - Access first row, third column (row index 0, column index 2) → expected value 100.
  - Learners were instructed to create a dictionary, build a DataFrame, and use iat to access a minimum of 5 individual values as a drill.

- Label-based access with DataFrame.loc and DataFrame.at:
  - loc: access by row label/index and column name(s); can return entire rows or specific columns.
  - at: similar to iat but uses label-based row and column references for accessing a single value.
  - Instructor noted that some methods overlap in outcome:
    - at and loc can produce the same result in many cases.
    - iat and iloc are similarly related (integer position based).

- Conditional access (boolean indexing) on DataFrame:
  - Condition is applied to one column (part 1). Based on that selection, either the full row is returned or a specific column’s value(s) are returned (part 2).
  - Examples illustrated:
    - Selecting rows where a column value > 10 and then returning a specific column value from those rows (e.g., returns 35 for a chosen example).
    - Using condition on column1 to then retrieve data from column2 for matching rows (example returned 200).
    - If only the condition is provided (no column specified), full matching rows are printed.
    - If condition matches no rows (e.g., >20 in that example), the result is empty (0 rows printed).
    - Selecting same column for both condition and returned data: e.g., column_name > 5 returns only values for that column that meet the condition.
  - Instructor highlighted the flexibility to mix condition column and returned column(s).
  - Drill: learners asked to produce at least 10 different combinations (part 1 conditions and part 2 retrievals) using the already created DataFrame. 20 minutes were allocated for this drill.
