## Inspection and Indexing Demonstrations
- head() and tail(): show top 5 and last 5 rows respectively.
- info(): shows column counts, data types, and null counts (in this dataset info() indicated no nulls and 4600 rows per column).
- describe(): used to compute descriptive statistics (count, mean, std, min, 25%, 50%, 75%, max) — instructor stressed descriptive statistics apply only to numeric columns.
- iloc usage and examples:
  - iloc[0] returns the first row (index 0) across all columns.
  - iloc[10] returns the 11th row (index 10) across all columns.
  - iloc can accept row index only, or row, column (row, col), and supports slice notation (start:stop) — stop is exclusive.
  - iloc default behavior targets rows unless columns are specified.
