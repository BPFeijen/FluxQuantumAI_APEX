## Discussion
### Focus on p-value as the critical statistic
* Geetika emphasized repeatedly that the p-value is the primary metric to focus on for both one-sample and two-sample t-tests and for Z-tests.
* Rule presented: if p-value < 0.05, the result is considered statistically significant; if p-value >= 0.05, further investigation is needed and no claim is made.
* She described this uniformly across test types: one-sample, two-sample, one-tailed, two-tailed, t-test and Z-test.

### Interpretation workflow (how to map p-value to hypothesis decision)
* First check p-value against alpha (0.05). If p < 0.05, there is a statistical difference; then consult the null hypothesis to determine whether to reject or fail to reject it.
* Examples: if null hypothesis states equality (means equal) and p < 0.05, reject the null; if null hypothesis states inequality and p < 0.05, fail to reject the null (i.e., the claim of difference is supported).
* If p >= 0.05, do not conclude — perform additional investigations.
