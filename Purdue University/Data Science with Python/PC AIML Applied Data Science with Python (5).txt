## Key topics and points covered

1. Array dimensionality and construction
- 0D, 1D, 2D and 3D arrays explained via nested lists:
  - Single value -> 0-dimensional array
  - Single list -> 1-dimensional array
  - [list1, list2] -> 2-dimensional array
  - Multiple such groups of lists (L1,L2 repeated) -> 3-dimensional array
- Visual labeling/examples to help learners remember dimension rules.

2. Element-wise arithmetic operations (NumPy arrays)
- Addition, subtraction, multiplication performed element-wise (index-wise pairing of A and B).
- Demonstrated on 1D and 2D arrays: each element at position (or index) in A corresponds to same index in B for the operation.

3. Division and division-by-zero behaviour
- Division is element-wise.
- When a divisor element equals 0 (denominator zero), an error occurs: "divide by zero encountered" and the operation fails for those positions.
- When a numerator element is 0, result is 0 (no error).

4. Exponentiation
- Power operation (A ** B) is element-wise: each element of A raised to the power of corresponding element of B.
- Example: 2**2, 2**3, 2**4, etc., producing 4, 8, 16, ...

5. Statistical functions (NumPy / built-in methods)
- Methods demonstrated/mentioned: median, mean, variance, standard deviation, percentiles, correlation.
- Example dataset used (3x3 2D array): [4, 3, 2; 10, 1, 0; 5, 8, 24]
  - Reported results: median = 4, mean ≈ 6.3, std ≈ 6.94, variance ≈ 48.2 (values shown by instructor).
- Important notes:
  - For 1D arrays, mean/median/variance calculations are straightforward (sum/ count etc.).
  - For 2D arrays, these statistical measures can be computed across the entire flattened dataset or along specified axes (rows/columns); the instructor emphasized that the mathematical formulas and axis-based computations will be covered in later statistics lessons (lessons 6 or 7).

6. Percentiles and quartiles
- Percentiles/quartiles partition a dataset based on their statistical formula (not just a simple mapping to 0-100 by intuition).
- Quartiles divide distribution into quarters (0-25, 25-50, 50-75, 75-100); percentiles generalize this.
- For 2D arrays, percentile calculation is more advanced and should be applied carefully (instructors will cover formulas in the statistics module); recommended to use provided library methods rather than guessing formulas.

7. String operations (NumPy add/standard string methods)
- Adding (NumPy add) on strings concatenates strings element-wise (concatenation / catenation was emphasized as a key term).
- Replacement: use replace method (specify full string, substring to replace, replacement substring) — example: replace "how" with "hi".
- Case conversions: upper() to convert to uppercase, lower() to convert to lowercase.

8. Indexing and negative indexing (1D, 2D, 3D arrays)
- Indexing conventions: indices start at 0 for forward indexing.
- Row index is denoted by i; column index by j. Example: element at first row, second column is (0,1) in zero-based indexing.
- 1D array: index 3 refers to the 4th item (0,1,2,3).
- 2D array: access [row_index, column_index], e.g., first row third element -> [0,2].
- 3D array: must first index the 2D sub-array (which 2D array you want: 0 or 1...), then row, then column. Example walk-through: accessing the second 2D array then first row first column returned value 7 in the example.
- Negative indexing counts from the end; negative indices start at -1 for the last element, -2 for second last, etc. Important clarification: negative indexing does not include 0 (0 is only for forward indexing).
- Examples illustrated: using one negative index in 2D/3D to select rows/columns from the end.
