1. `HedgeManager.process()` detecta pullback.

3. Log file escrito (comportamento original).

5. Canonical payload emitido → `tg.notify_decision()` → Barbara recebe **↩ PULLBACK_START — <hedge_dir>** (icon map Fase 2 M6.3).

1. Pullback hedge fecha em profit (trend voltou).

3. **[NEW]** map → `"PULLBACK_END_EXIT"` → Telegram **↪ PULLBACK_END_EXIT**.
